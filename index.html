Recent Projects

Project 1

JuriesGenie

JuriesGenie is a RAG-based tool designed to scrape the Kanoon website using BeautifulSoup, chunk the data, generate embeddings, and store them in a vector database. Queries are processed similarly, with their embeddings matched against the database for similar vectors. Results are displayed via a Flask UI, accessible online using Ngrok. The system integrates web scraping, semantic chunking and vector search for legal information retrieval.

Project 2

Library Catalog System

The Library Catalog System is a command-line application developed in C for managing a library's book inventory. It allows users to add, remove, search for, and display books using basic text-based menus. The system uses structures for book data and file handling for data persistence.

Project 3

OOP Email System

The Email System is an object-oriented program in C++ designed to manage sending, receiving, and organizing emails. It utilizes classes to represent users, emails, and folders, allowing for features like composing, reading, and deleting messages. The system supports basic functionalities such as filtering and searching emails by sender or subject.

Project 4

Chat application

The Chat Application in Java is a real-time messaging platform that allows users to communicate through text messages. Utilizing socket programming, it establishes a client-server architecture for sending and receiving messages over the network. The application features user authentication, message broadcasting, and a simple graphical user interface (GUI) with swing for an enhanced user experience.

Project 5

Car dealership database

The Car Dealership Database is a management system designed to store and organize information about vehicles, customers, and sales transactions. It includes features for adding, updating, and retrieving data on car inventory, customer details, and sales records.
